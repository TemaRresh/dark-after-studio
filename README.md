# Dark After Studio

Первый прототип браузерного генератора стилизованных изображений для мерча и айдентики Dark After.

## Возможности

- Загрузка фотографии локально, без отправки на сервер
- Живое большое превью
- Автоматические миниатюры всех стилей
- ASCII Mono
- ASCII Color
- Lyric Field
- Binary
- Halftone
- Broken Scan
- Настройки детализации, контраста, яркости, цветов и прозрачности
- Экспорт PNG
- Адаптивный интерфейс для компьютера, iPad и телефона

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## GitHub Pages

В настройках репозитория выберите Pages, Deploy from a branch, ветка `main`, папка `/root`.
